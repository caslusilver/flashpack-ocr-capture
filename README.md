# FlashPack OCR Capture

Plugin WordPress para captura de etiquetas, crop assistido e envio para OCR via webhook.

## Recursos
- Captura com molduras (verde + vermelha)
- Crop preciso baseado na moldura
- Envio automático ao webhook
- Modal com dados extraídos (nome, bloco, ap, CEP)
- Pronto para atualização via GitHub Updater

## Instalação
1. Baixe o ZIP do repositório.
2. Envie para **Plugins > Adicionar novo > Enviar plugin**.
